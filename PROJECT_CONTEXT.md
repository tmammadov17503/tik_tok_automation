# Project Context

This workspace was created to hold the conversation output for a new short-form video workflow project.

Primary goal:

- Build a local "start button" app that automates as much of the short-video pipeline as possible.

Current direction:

- Replace paid hosted editing steps with local free/open-source tools.
- Keep the workflow visual and simple, closer to an n8n-style board than a raw script.
- Export ready-to-review vertical clips, metadata, and run logs.
- Avoid watermark removal and other unsafe shortcuts.

Current entrypoint:

- `python app.py`
