Sorted Space Agent

Privacy Policy

Last updated: June 16, 2026

Overview

Sorted Space Agent is a private automation tool for The Sorted Space. It helps an authorized account owner create and schedule original Pinterest Pins from approved affiliate product links.

Information we process

Depending on configuration and permissions granted by the account owner, the app may process:

  • Pinterest account identifiers, board IDs, Pin IDs, and OAuth tokens
  • Amazon Associates product links provided by the account owner
  • generated Pin titles, descriptions, image prompts, images, and posting schedules
  • Telegram admin chat identifiers used for queue control and status notifications
  • technical logs needed to troubleshoot publishing jobs or failed API requests

How information is used

  • to authenticate API requests authorized by the Pinterest account owner
  • to create Pins on selected Pinterest boards
  • to generate original visual and written content for approved product links
  • to maintain queue status, scheduling, confirmations, and troubleshooting logs
  • to include required affiliate disclosures in Pin descriptions

Storage

Tokens, queue data, generated images, and logs may be stored in the operator's private local or server environment. The app does not require users to share Pinterest passwords. Secrets should be stored only in protected configuration files.

Sharing

Information is shared only as needed with services configured by the operator, including Pinterest for Pin publishing, OpenAI or another configured image provider for content generation, Telegram for admin notifications, and Amazon as the linked destination when viewers click an affiliate link.

User responsibilities

The operator is responsible for submitting only links and content they are authorized to promote and for complying with Pinterest policies, Amazon Associates requirements, advertising disclosure rules, and applicable law.
